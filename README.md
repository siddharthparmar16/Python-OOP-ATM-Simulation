# Python OOP ATM Simulation

This is a beginner-level ATM simulation project created while learning Object-Oriented Programming in Python.

## Features
- Create PIN
- Change PIN
- Check balance
- Deposit money
- Withdraw money
- Exit menu

## Concepts Practiced
- Classes and objects
- Constructor `__init__`
- Instance variables
- Private attributes
- Methods
- Conditional logic
- Menu-based program flow

## Future Improvements
- Add 3 wrong PIN attempts
- Add transaction history
- Prevent negative deposits/withdrawals
- Support multiple users/accounts
- Store account data in a file or database
